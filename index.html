FAQ

 Q: How to install applications for the homebrew launcher? A: find the .3dsx and .sdmh files and put them both in a folder (there may not be a .sdmh file) then put that folder in /3ds/ on your sd card

Q: Help! I cant boot without my sd!
A: get godmode9 and find arm9loaderhax.bin on the sd then put arm9loaderhax.bin on the SysNand CtrNand

Q: Can I update to the latest version with A9LH?
A: run luma updater and hit normal. click start than update through system settings as normal.

Q: Sighax? How? What?
A: We still need the bootrom for sighax, so it's not getting released soon.

Q: What will Sighax do?
A: Sighax will not:
  • Allow installation of A9LH on 11.3
  • Allow further editing of the system from A9LH
A: Sighax will:
  • Allow installation of A9LH with only arm9 access
  • Make downgrading obsolete
  • Pave the way to A9LH installation on 11.3
  • Allow a cleaner implementation of A9LH that does not require corrupting the FIRM
